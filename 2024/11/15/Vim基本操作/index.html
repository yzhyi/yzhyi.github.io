Vim基本操作

发表于 2024-11-15 更新于 2024-12-12

操作

基础操作

  • f{char}

    • 使用;键可以跳到下一个
    • 使用,键可以跳到上一个

  • %s/word1/word2/g: 全局替换

  • gv: 重选上一次可视模式

  • 模式下,r可以修改所有列选中的位置

  • !{cmd}: 运行shell命令

  • ea: 在当前单词结尾后添加

  • m{a-zA-Z} `{mark}: 添加标签,回到标签(大写字母是全局标记)

  • ``: 当前文件中上次跳转动作之前的位置

    • %r]``r[: 修改包围的符号,change surround

  • `.: 上次修改的地方

  • ^: 移动光标到行首第一个非空字符处

  • $: 移动光标到行末最后一个非空字符处

  • =: 自动缩进(格式化)

  • dtn: 删除字符 n 前面的内容

  • zz: 将光标所在行移动到窗口中央

  • 撤销在普通模式下使用 u，反撤销使用 Ctrl+r

  • Ctrl + a 来在数字文本上进行递增操作；递减操作使用 Ctrl x 。（如果光标不在数字所在位置，该命令会在本行往后查找第一个出现数字的地方，并将光标跳转到该处，然后执行加1的操作。）

  • <Ctrl + h>: 删除光标前一个词(与退格键相同)（插入模式下）

  • <Ctrl + w>: 删除光标前一个单词（插入模式下）

  • <Ctrl + u>: 删除至行首（插入模式下）

  • gv: 选中上次选中的区域

  • ctrl+o：回到上一次跳转的地方

  • 对比两个文件:

    • 在终端下: nvim -d 文件名1 文件名2
    • 在nvim中: :diffsplit 要对比的文件名

  • gi: 跳转到上次插入的地方

  • ctrl+z暂停vim返回shell,shell里做完想做的事,然后fg返回vim

.命令

  • .命令会重复上次修改
    • 重复normal模式下的修改操作(dd,x,>>)
    • 从进入insert模式的那一刻时(包括进入的命令),直到返回normal模式时为止

寄存器

  • “{register}指定要用的寄存器
  • “+ X11剪切板
  • “* X11剪切板

宏

  • q{register}….q: 录制宏
  • @{register}: 回放宏
  • 当动作命令失败时,宏将终止执行
  • q{大写字母}: 追加宏操作

替换

  • [range]s[ubstitute]/{pattern}/{string}/[flags]
  • range:
    • 使用可视模式选取范围.
    • %: 表示整个文件.
    • 1,20: 第1行~第20行.
    • .,+2 当前行~往后2行.
    • .,$ 当前行~文件尾.
  • flags:
    • g: 全局范围内执行
    • c: 可以确认或拒绝每一处修改
    • n: 抑制正常的替换行为,只是报告本次substitute命令匹配的个数
